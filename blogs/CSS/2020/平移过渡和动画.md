## transform 平移 缩放 转动 拉长或拉伸

## transform 2D转换

```html
<style>
    .box{
        width:200px;
        height:200px;
        margin:100px auto;
        background-color: aquamarine;
    }
    .box:hover{
        /** translate 水平和垂直位置的平移 */
        transform:translate(30px,30px);
        /** rotate 2D 角度 正--顺时针 */
        transform:rotate(30deg);
        /** scale 缩放宽高 倍数 */
        transform:scale(1,3);
        /** skew x y轴倾斜拉伸 */
        transform:skew(20deg,30deg);
    }
</style>
```

## transform 3D转换

```html
<body>
    <div class="father-box">
        <div class="box">
        </div>
        <div class="box">
        </div>
        <div class="box">
        </div>
    </div>
</body>
<style>
.father-box{
    width:500px;
    height:500px;
    margin:100px auto;
    /** 设置透视视图 实现3D效果必须属性 距离越小越明显 */
    perspective: 800px;
    /** 视线位置 */
    perspective-origin: center center;
    /** 设置3D效果 */
    transform-style: preserve-3d;
}
.box{
    width:200px;
    height:200px;
    background-color: red;
    margin: 20px;
}
.box:nth-child(1):hover{
    transform:rotateX(10deg) translateZ(100px)
}
.box:nth-child(2):hover{
    transform:rotateY(40deg) translateZ(-100px)
}
.box:nth-child(3):hover{
    transform:rotateZ(40deg)
}
</style>
```

## transition过渡

```html
<style>
.box{
    width:200px;
    height:200px;
    background-color: red;
    margin: 20px;
    /** 设置过渡的CSS属性 */
    transition-property:background-color;
    transition-property:all;
    /** 设置完成时间时间 秒/毫秒 */
    transition-duration: 2s;
    /**过渡效果的转速曲线  aes:慢-快-慢；ease-out 慢速结束 ease-in慢速开始 ease-in-out 慢-快-慢 linear 匀速*/
    transition-timing-function: ease-in;
    /** 开始的时间，可以延迟几秒后执行 默认为0 */
    transition-delay: 0s;
    /** 综合属性 */
    transition:all 2s linear 2s;
}
.box:hover{
    background-color: #000;
    margin-left:500px;
}
</style>
```

## 动画

```html
<style>
.box{
    width:200px;
    height:200px;
    margin: 100px auto;
    /** 动画名字 */
    animation-name:change;
    /** 动画执行时间 */
    animation-duration:2s;
    /** 动画曲线，如何完成一个周期 */
    animation-timing-function:linear;
    /** 延迟开始 */
    animation-delay:2s;
    /**动画的播放次数 默认1次 infinite 无穷次 */
    animation-iteration-count: infinite;
    /* 是否循环交替反向播放动画 normal 正常播放/reverse 反向播放/alternate 往返播放*/
    animation-direction:alternate;
}
.box:hover{
    background-color: #000;
    margin-left:500px;
}
/* @keyframes animationname {keyframes-selector {css-styles;}} */
@keyframes change{
    /* 0-100%
    from (和0%相同)
    to (和100%相同) */
    from{
        /* 一个或多个合法的CSS样式属性 */
        background-color: red;
    }
    to{
        background-color: #000;
    }
    /** 自定义 */
    0%{   }
    25%{  }
    75%{    }
}
/**   比如加入购物车 小球的轨迹*/
.box{
    width:200px;
    height:200px;
    margin: 100px auto;
    /** 综合属性*/
    animation:move 2s linear infinite;
    background-color: aquamarine;
}
@keyframes move{
    from{
        transform: rotate(0deg);
    }
    to{
        transform: rotate(300deg);
    }
}
</style>
```

## 爱心的实现

1.旋转矩形
2.两个圆形

```html
<style>
    .heart {
        width: 200px;
        height: 200px;
        background-color: pink;
        margin: 100px auto;
        position: relative;
        transform:rotate(-45deg);
        /** alternate 往返运动 */
        animation:move 0.8s infinite alternate;
    }
    @keyframes move{
        from{
            transform:rotate(-45deg) scale(0.5)
        }
        to{
            transform:rotate(-45deg) scale(1)
        }
    }
    .heart::after, .heart::before {
        content: "";
        display: block;
        width: 100%;
        height: 100%;
        background-color: pink;
        border-radius: 50%;
        position: absolute;
    }
    /** left优先级高于right */
    .heart::before {
        top: -50%;
        left: 0;
    }
    .heart::after {
        right: -50%;
        top: 0;
    }
</style>
```
